asynchronous server of second layer: 
initialization: {
ip_1, port_1 // 作为server被worker连接的ip和port
ip_2, port_2 // 作为client连接第三层的server的ip和port
w, b         // 模型参数，全局变量
}

while True: {
	listen server address(ip_1, port_1)
	
	start a new thread: // 当有一个client连接server时，开启一个新的线程处理这个client与server的消息
		send message to the server of third layer
		receive message from server of thrid layer
		send message to client 
}
